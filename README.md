# MLBetterCode (BetterCode)

Клиентский мод под **Minecraft 1.12.2 (Forge)** для Mineland Creative+: ускоряет “печать” кода, кеширует GUI/сундуки и даёт команды-утилиты для разработчика.

## Требования

- Minecraft `1.12.2`
- Forge `1.12.2-14.23.5.2859+`
- Java `8`

## Установка

1) Скачай `bettercode-*.jar` из **Releases** этого репозитория.
2) Положи в `.minecraft/mods/`
3) Перезапусти игру.

## Команды (основное)

- `/mldsl run [path] [--start N]` — запустить `plan.json` (печать через `/placeadvanced`).
- `/placeadvanced ...` — печать блоков/действий + заполнение сундука параметров.
- `/tppath <x> <y> <z>` — перелёт по “шагам” (безопасно для Dev+Creative).
- `/cacheallchests [stop]` — открыть и закешировать сундуки рядом/по зоне.
- `/regallactions [stop]` — обход меню действий и кеширование (экспорт в `.minecraft/regallactions_export.txt`).

## /copycode (перенос кода между участками)

Команды работают в **Dev + Creative**.

- `/copycode <id1> <id2> <yoffset> <floorsCSV>`
  - `id1` — исходный участок (куда вернуться после копирования)
  - `id2` — целевой участок
  - `yoffset` — на сколько сместить Y при постановке на `id2` (пример: `10`)
  - `floorsCSV` — список этажей относительно синего стекла: `0,10,20` и т.п.
- `/cancelcopy` — остановить перенос

Поведение: мод сканирует этажи по синим стёклам, находит все блоки над ними (кроме воздуха), летит к каждому (tppath на `z+2`), ломает `Shift+ЛКМ`, переключается на `id2`, ставит маяк на те же координаты с `+yoffset`, возвращается на `id1` и продолжает. В actionbar показывается прогресс и примерный ETA.
