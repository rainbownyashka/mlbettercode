# MLBetterCode — гайд (RU)

Это “пользовательский” гайд по модулю печати/ввода. Подробности реализации лежат в коде.

## Команды

### `/mldsl`

- `/mldsl run [path] [--start N]`
  - `path` — путь к `plan.json` (если не указан, используется `.minecraft/plan.json`)
  - `--start N` — начать с записи `N` (0‑based)
- `/mldsl check [path] [--start N]`
  - сканирует уже существующий код и пишет в чат “FOUND/NOT found”

### `/placeadvanced`

Низкоуровневая команда “поставь блок → открой меню → выбери действие → (если нужно) заполни сундук параметров”.

Формат:

`/placeadvanced <block> <name> <args|no> [<block> <name> <args|no> ...]`

Внутри `args`:
- `slot(N)=text(...) / num(...) / var(...) / var_save(...) / arr(...) / arr_save(...) / loc(...)`
- `slot(N)=item(stone, count=3, name="&a...", description="l1\\n&bL2", nbt={})`
- `clicks(N,K)=0` — клики по слоту (enum‑переключатели)

### `/tppath`

`/tppath <x> <y> <z>` — построить очередь “TP‑пути” к координатам (в Dev+Creative).

### `/cacheallchests`

Кеширование сундуков вокруг (используется для валидации/быстрого чтения):

- `/cacheallchests` — поставить в очередь все загруженные “целевые” сундуки
- `/cacheallchests stop` — остановить

### `/regallactions`

Кеш действий (проход по GUI, сохранение в `.minecraft/regallactions_export.txt`):

- `/regallactions [stop]`
- `/regallactionsdebug [on/off]`

## Хоткеи (Dev+Creative)

- `R` или `Right Alt` — открыть code menu
- `G` — TP forward
  - удерживай `G` и крути колесо — задаёт количество шагов
  - отпусти `G` — выполнит серию шагов
  - пока удерживаешь `G`: `+` поднимает на `+10y`, `-` опускает на `-10y`

## Ввод через “стёкла” (GUI‑input)

Тип ввода определяется **панелью‑маркером** (`stained_glass_pane`):
- клик по стеклу → мод ищет соседний слот (вниз → влево → вправо → вверх)
- если рядом уже лежит валидный предмет нужного типа → открывает редактирование
- иначе берёт пустой слот и кладёт туда предмет‑значение

Управление вводом:
- `Enter` — применить
- `Shift+Enter` — применить + “extra stack” (если поддерживается)
- `Tab` — вставить подсказку (suggestion)
- `Shift+Tab` — применить (как Enter)
- `Esc` — отмена

Также есть side‑панели “recent” (недавние) и “frequent” (частые): кликом можно подставлять значение в текущее редактирование.
