# AGENT_TAG_INDEX

Purpose: searchable map of stable `[TAG:...]` comment families used in code.

## Tag Families
- `rga-*`
Description: regallactions / resave2 flow, menu clicks, page save/merge, validation.

- `chest-page-*`
Description: chest page detection, next-page click, cursor wait, merge state.

- `publish-*`
Description: `/module publish` warmup/open/page stabilization/close behavior.

- `export-*`
Description: export pipeline for json/txt outputs and record assembly.

- `module-*`
Description: module load/confirm/apply flows and hub integration touchpoints.

## Rule
When introducing a new non-trivial logic area, add a new family here.
